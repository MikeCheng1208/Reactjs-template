# webpack for Reactjs template
- 使用reactjs跟react-router
- 可以使用javascript es6
- 使用stylus & sass & scss 寫 css

* 啟動使用 npm run dev
```
npm run dev
```
* 編譯使用 npm run deploy  or  webpack -p
```
npm run deploy

or

webpack -p
```
